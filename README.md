# wistem

This repository contains data and code.

  - Heidi Blackburn and Jason Heppler, “Women in STEM in Higher
    Education: A Citation Analysis of the Current Literature,” *Science
    & Technology Libraries* 38:3 (Fall 2019). DOI:
    [10.1080/0194262X.2019.1645080](https://doi.org/10.1080/0194262X.2019.1645080)
  - Heidi Blackburn and Jason Heppler, “Who Is Writing About Women in
    STEM in Higher Education in the United States? A Citation Analysis
    of Gendered Authorship,” *Frontiers in Psychology* 10 (Spring 2020).
    DOI:
    [10.3389/fpsyg.2019.02979](https://doi.org/10.3389/fpsyg.2019.02979)
